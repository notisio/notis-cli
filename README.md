# notis-cli

Public release binaries for the **Notis** endpoint agent (`notis`) and the bundled gateway (`notis-gateway`).

The source lives in a private repository; this repo only hosts downloadable release assets.

## Install

- **macOS / Linux (Homebrew):** `brew install notisio/tap/notis`
- **Windows (Scoop):** `scoop bucket add notis https://github.com/notisio/scoop-bucket` then `scoop install notis`
- **curl / iwr one-liners:** see https://notis.io/download

Releases are produced automatically by GoReleaser. See https://notis.io
